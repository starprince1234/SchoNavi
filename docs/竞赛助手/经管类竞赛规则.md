## 二、调研与案例类

### 全国大学生市场调查与分析大赛

- 先确定总体、抽样框和研究问题，再设计问卷；不能先收方便样本再包装结论。
- 报告应说明预测试、信效度、响应率、缺失值、权重、推断边界和隐私保护。
- 商业建议必须由数据支持；相关性不能直接写成因果关系。

### 全国大学生物流设计大赛

- 通常围绕指定企业案例，重点是问题诊断、数据口径、网络/库存/运输模型、落地路径和经济性。
- 企业材料可能受保密限制，不得把赛题数据上传公开仓库或第三方 AI。
- 方案需与企业实际约束一致，不要只给脱离资源和组织条件的理论最优解。

### 税收风险管控与会计案例

法规版本、税种、会计准则和业务期间必须对应。引用法条时记录施行日期；不使用已废止政策，也不把竞赛分析当作真实企业税务意见。

## 三、创业与商业策划类

### 三创赛

常规赛与实战赛的评价逻辑不同。常规赛强调创新、创意、创业方案和答辩；实战赛可能要求真实平台任务、运营数据或阶段排名。订单、流水、用户和投放数据必须真实可核验。

### 全国高校商业精英挑战赛

这是系列赛事。品牌策划、文旅与会展、国际贸易、创新创业、会计与商业管理案例等子赛的报名、队伍和材料不同，不能共用一套时间表或评分表。

### “工行杯”金融科技创新大赛

处理金融数据和场景时要关注隐私、授权、算法公平、可解释性、模型风险和监管约束。演示原型不得接入真实账户或使用未脱敏客户数据。

## 四、经营模拟类

“学创杯”、全国企业竞争模拟大赛和数智化企业经营沙盘大赛是不同赛事/平台，成绩和奖项不能互换。

共同训练重点：

- 现金流优先于纸面利润，避免资金链断裂。
- 明确市场、产能、库存、融资和研发的滞后关系。
- 每轮决策前做情景预算，赛后做偏差分析。
- 记录参数和决策理由，不靠“多开账号试错”等违规方式获取信息。

## 六、经管成果证据标准

|材料|合格证据|常见问题|
|---|---|---|
|市场规模|权威统计、年份、地区、口径和推导|直接复制咨询图，不说明口径|
|用户需求|抽样/访谈记录、脱敏原始数据|只放几张聊天截图|
|收入订单|合同、发票/回款、后台导出并脱敏|把意向、GMV 当收入|
|成本利润|完整成本、税费、获客和履约口径|只算原材料，不算人工渠道|
|预测|假设、情景、敏感性和现金流|单一路径高速增长|
|政策法规|正式文件、文号、版本和施行日期|引用过期政策或二手解读|

## 七、提交前检查

- [ ] 数据和案例来源合法，商业秘密已脱敏。
- [ ] 统计方法与样本设计匹配，结论不过度外推。
- [ ] 财务三表/关键指标口径一致，单位和期间一致。
- [ ] 法规、税率、准则和行业数据为适用版本。
- [ ] 团队能解释每轮模拟决策和失败结果。
- [ ] 不虚构客户、合同、收入、融资或合作证明。
