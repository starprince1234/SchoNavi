## 数据口径

1. “入选竞赛分析报告目录”不等于“教育部官方认证”或“所有学校均认定为同一等级”。《全国普通高校大学生竞赛分析报告》由中国高等教育学会高校竞赛评估与管理体系研究专家工作组发布，学校的综测、推免、奖学金和经费政策应以本校文件为准。
2. “国家级、省级、校级”是具体赛事和学校管理中的认定口径，不能只凭赛事名称中的“全国”判断。
3. 不承诺获奖能带来保研、Offer、奖金或实习绿色通道。此类权益只有当届官方文件或学校制度明确写明时才成立。
4. 不把往届获奖比例外推为当届概率。参赛规模、赛区名额、评审规则和奖项数量都会变化。
5. 无法从官方页面核验的商业宣传、培训机构排名和“含金量榜单”不作为事实来源。

## 信息核验顺序

1. 当届赛事官网的通知、章程、参赛手册和 FAQ。
2. 主办单位官网或教育行政部门通知。
3. 省级赛区、承办高校和本校教务处/创新创业学院通知。
4. 聚合报名平台和自媒体只用于发现线索，不用于确认规则。
