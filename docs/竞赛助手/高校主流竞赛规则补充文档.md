# 高校主流竞赛规则补充文档

# 一、标杆赛事专项档案（2024-2025年度）

## 1.1 “挑战杯”全国大学生课外学术科技作品竞赛（大挑）

### 1.1.1 官方章程摘要

主办单位：共青团中央、中国科协、教育部、全国学联、省级人民政府；参赛对象：全日制普通高等学校在校专科生、本科生、硕士研究生、博士研究生（不含在职研究生）；竞赛主题：崇尚科学、追求真知、勤奋学习、锐意创新、迎接挑战；赛道设置：自然科学类学术论文、哲学社会科学类社会调查报告和学术论文、科技发明制作（A类：产品已完成研发和中试，B类：产品仅为实验室阶段或创意构想）。

### 1.1.2 时间节点

- 校赛：2024年10月-11月（校内报名、评审、推荐）

- 省赛：2025年3月-4月（材料提交、省级评审、公示）

- 国赛初评：2025年5月-6月（全国组委会资格审查与初评）

- 国赛决赛：2025年10月（现场展示、答辩、评审）

### 1.1.3 评审标准

|评审维度|权重|核心要求|
|---|---|---|
|创新性|30%|选题新颖，研究方法创新，观点独特，技术方案有突破|
|科学性|25%|理论依据充分，实验设计合理，数据真实可靠，逻辑严密|
|实用性|25%|贴近社会需求，有实际应用价值，可转化性强|
|规范性|20%|论文/报告结构完整，格式规范，语言表达准确，参考文献引用规范|
## 1.2 “挑战杯”中国大学生创业计划竞赛（小挑）

### 1.2.1 官方章程摘要

主办单位：共青团中央、教育部、人力资源社会保障部、中国科协、全国学联、省级人民政府；参赛对象：全日制普通高等学校在校学生（含专科生、本科生、研究生）和毕业5年内的高校毕业生（2020年及以后毕业）；赛道设置：高教主赛道（本科生、研究生组）、“青年红色筑梦之旅”赛道（公益组、商业组）、职教赛道、国际赛道。

### 1.2.2 时间节点

- 校赛：2025年3月-4月（校内报名、孵化、评审）

- 省赛：2025年6月-7月（材料提交、省级复赛与决赛）

- 国赛网评：2025年8月-9月（全国网评筛选入围决赛项目）

- 国赛决赛：2025年11月（现场路演、答辩评审）

### 1.2.3 评审标准

|评审维度|权重|核心要求|
|---|---|---|
|项目创新性|25%|产品/服务有核心技术或独特模式，竞争壁垒明显|
|商业可行性|30%|市场分析准确，商业模式清晰，盈利预测合理，运营计划可行|
|团队能力|20%|团队结构合理，成员互补性强，有相关行业经验或资源|
|社会价值|15%|带动就业、助力乡村振兴、促进社会可持续发展等贡献|
|路演表现|10%|表达清晰流畅，PPT逻辑严谨，答辩应变能力强|
## 1.3 中国国际“互联网+”大学生创新创业大赛

### 1.3.1 官方章程摘要

主办单位：教育部等12部委和省级人民政府；参赛对象：普通高等学校、职业院校学生（全日制在校生、毕业五年内毕业生、港澳台地区及国际学生）；竞赛主题：我敢闯，我会创；赛道设置：高教主赛道（本科生、研究生、毕业五年内校友组）、“青年红色筑梦之旅”赛道（公益组、商业组、创意组）、职教赛道、国际赛道、产业命题赛道（与龙头企业合作命题）。

### 1.3.2 时间节点

- 校赛：2024年5月-6月（校内报名、培训、评审）

- 省赛：2024年7月-8月（省级复赛、决赛及推荐）

- 国赛网评：2024年9月（全国网评确定入围决赛名单）

- 国赛决赛：2024年10月（现场总决赛）

### 1.3.3 评审标准

|评审维度|权重|核心要求|
|---|---|---|
|创新性与引领性|25%|技术/模式创新程度高，引领行业发展趋势，拥有自主知识产权|
|商业价值|30%|市场规模大，增长潜力足，盈利模式可持续，已产生或预期经济效益好|
|团队综合素质|20%|团队成员背景匹配，执行力强，有核心领军人物|
|社会价值与责任|15%|助力国家战略，解决社会痛点，环保节能，促进共同富裕|
|现场展示与答辩|10%|项目陈述重点突出，PPT设计专业，答辩逻辑清晰、反应迅速|
## 1.4 国家级大学生创新创业训练计划项目（大创项目）

### 1.4.1 官方章程摘要

主管单位：教育部；实施主体：各高校；项目类型：创新训练项目（本科生个人或团队，开展创新性研究）、创业训练项目（本科生团队，开展创业模拟）、创业实践项目（本科生团队，依托企业开展创业实践）；资助等级：国家级、省级、校级，资助经费按等级发放。

### 1.4.2 时间节点

- 申报立项：2024年3月-4月（校内申报、评审、推荐；省级/国家级评审）

- 中期检查：2024年11月-12月（项目进展汇报、阶段性成果提交）

- 结题验收：2025年4月-5月（提交结题报告、成果证明，校内验收；省级/国家级抽查）

### 1.4.3 评审标准

|评审维度|权重|核心要求|
|---|---|---|
|项目选题|25%|选题具有科学性、创新性和实用性，符合学科发展趋势或社会需求|
|研究/实施方案|30%|技术路线清晰，方法可行，进度安排合理，资源保障充分|
|团队与指导教师|20%|团队成员分工明确，指导教师专业背景匹配，指导经验丰富|
|预期成果|25%|成果目标具体可衡量，学术价值或应用价值高，可实现性强|
# 二、历年规则变化对照表（近三届）

## 2.1 “挑战杯”大挑（2021-2025届）

|年份/届数|核心变化|影响|
|---|---|---|
|2021年（16届）|哲学社科类作品新增“乡村振兴”专项主题；科技发明制作A类需提供第三方检测报告|引导选题聚焦国家战略；提高成果转化门槛，增强项目真实性|
|2023年（17届）|参赛对象排除在职研究生；自然科学类论文限报3篇/校|保障在校学生参赛权益；提升校内选拔竞争力，优中选优|
|2025年（18届）|新增“科技伦理审查”环节；国赛初评增加网络公示环节|强化科研伦理意识；增加透明度，杜绝违规项目|
## 2.2 “互联网+”大赛（2022-2024届）

|年份/届数|核心变化|影响|
|---|---|---|
|2022年（8届）|增设产业命题赛道；“青红旅”赛道分公益组、商业组|促进产学研融合；细化赛道评审标准，针对性更强|
|2023年（9届）|毕业五年内校友组纳入高教主赛道；国际赛道参赛资格放宽至全球高校|鼓励校友创业；提升赛事国际化水平，促进中外交流|
|2024年（10届）|产业命题赛道合作企业增至50家；评审权重中“社会价值”提升5%|拓展项目落地资源；引导项目更关注社会贡献|
## 2.3 大创项目（2022-2024年）

|年份|核心变化|影响|
|---|---|---|
|2022年|创业实践项目需提供与企业合作协议；资助经费向理工科倾斜|确保创业实践真实性；鼓励理工科创新研究|
|2023年|中期检查增加“进展预警”机制；结题成果需体现知识产权|加强项目过程管理；提升成果质量，鼓励专利申请|
|2024年|创新训练项目允许跨校组队；校级项目资助比例提高至60%|整合校际资源；扩大覆盖面，鼓励更多学生参与|
