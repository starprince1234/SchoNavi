# 医学与生命科学类竞赛规则知识库

> 最近核验：2026-06-30。医学与生命科学项目首先受伦理、生物安全、数据保护和临床合规约束，竞赛要求不能替代这些要求。

## 一、核心赛事

|赛事|方向|典型形式|官方入口|
|---|---|---|---|
|中国大学生医学技术技能大赛|临床医学与医学技术技能|分专业/赛道的理论和技能竞赛|[教育部](https://www.moe.gov.cn/)（检索当届通知）|
|全国大学生生命科学竞赛（CULSC）|生命科学研究与创新创业|研究项目、论文/报告、展示和答辩|[CULSC](https://www.culsc.cn/)|
|全国大学生基础医学创新研究暨实验设计论坛（大赛）|基础医学、实验设计|研究项目、实验设计、壁报/答辩|[官网](http://www.jcyxds.com/)|
|全国大学生化学实验创新设计大赛|化学生物交叉实验|实验方案、数据、展示和答辩|[官网](https://cid.nju.edu.cn/)|

前三项列入 2024《全国普通高校大学生竞赛分析报告》目录；化学实验创新设计也在目录中，适合药学、化学生物学和相关交叉项目按当届资格参赛。

## 二、中国大学生医学技术技能大赛

- 由教育部组织的医学教育赛事，具体专业赛道、参赛院校和届次安排通过教育部及承办高校发布。
- 常见方向可能覆盖临床医学、护理学、中医学、预防医学等，但不能把某届赛道直接复制为下一届固定设置。
- 参赛资格通常由学校组织选拔，不是学生个人在商业平台直接报名。
- 技能训练必须遵循标准操作、患者安全、无菌原则、人文沟通和团队协作。

## 三、全国大学生生命科学竞赛（CULSC）

- 科学探究类与创新创业类的周期、材料和评审重点不同，进入对应赛道通知查询。
- 科学探究项目应有明确问题、预注册/方案、原始记录、统计设计、阴性结果和可复现过程。
- 创新创业项目除技术外，还要说明产品路径、法规分类、质量体系、知识产权和应用证据。
- 多人团队应能证明成员实际参与，导师课题成果不能直接包装为学生独立成果。

## 四、基础医学创新研究暨实验设计论坛（大赛）

重点通常包括科学问题、实验设计、创新性、技术路线、数据/预期结果和现场表达。准备时应区分：

- 已完成研究：提交真实原始数据、统计和重复实验。
- 实验设计：明确尚未验证的假设，不把预期结果写成已证实结论。
- 转化项目：说明动物、临床、注册和生产之间仍需经过的验证阶段。

## 五、伦理与安全

### 人体研究

- 涉及患者、健康数据、人体样本、影像或干预时，先确认伦理审批和知情同意。
- 公开数据集也要遵守数据使用协议，不尝试重新识别个人。
- 竞赛展示使用最小必要数据；病例、影像和基因数据充分脱敏。

### 动物与生物实验

- 动物实验需要相应伦理审批和 3R 原则说明。
- 病原体、转基因、生物样本和化学品按实验室生物安全与危化品制度管理。
- 不在不具备资质的场所开展培养、感染、动物或人体相关实验。

### 医疗 AI

- 数据按患者级切分，避免同一患者图像进入训练和测试。
- 说明阳性率、设备/中心差异、缺失数据和外部验证。
- 同时报告敏感度、特异度、置信区间和校准，不只报告准确率。
- 不把回顾性模型结果写成临床有效或可直接诊断；说明适用人群和失败风险。

## 六、证据分级

|证据|能支持的表述|不能越界到|
|---|---|---|
|体外实验|特定条件下的细胞/分子效果|人体疗效|
|动物实验|动物模型中的机制或效果|临床安全有效|
|回顾性数据|关联和预测性能|前瞻性临床效用|
|单中心前瞻研究|该中心和样本下的表现|广泛人群普适性|
|多中心临床研究|研究方案定义范围内的证据|未获批适应证宣传|

## 七、提交前检查

- [ ] 伦理批件、知情同意和生物安全审批与项目一致。
- [ ] 原始记录、样本编号、排除标准和统计脚本可追溯。
- [ ] 图像未选择性裁剪，凝胶/显微图处理有说明。
- [ ] 重复次数区分技术重复和生物学重复。
- [ ] 结论没有从相关性越界到因果、从实验越界到临床。
- [ ] 药品、器械、诊断和健康建议表述符合监管边界。
- [ ] 个人和患者信息已脱敏，外部数据许可可核验。
