🛠️ 学习平台与实用工具
除了比赛官网，同学们备赛时还需要各类辅助工具。以下工具能帮助团队提高协作和学习效率：

在线协作与项目管理：

腾讯文档/飞书文档：用于多人实时在线撰写和修改商业计划书、报告，支持评论和版本历史。

Trello/飞书项目：适合用看板管理备赛任务、设置截止日期并分配负责人，可视化追踪进度。

学习资源与技能提升：

中国大学MOOC(慕课)：可查找与比赛相关的专业课程（如商业分析、Python编程）来补足知识短板。

Bilibili（B站）：有许多技术博主、往届参赛者分享的竞赛经验、技术教程和项目复盘视频。

设计制作：

Canva/稿定设计：提供大量PPT、海报、展板模板，能快速制作高质量的路演材料和宣传图。
