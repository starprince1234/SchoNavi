# 机器人与人工智能类竞赛规则知识库

> 最近核验：2026-06-30。机器人赛事的组别、平台和器材变化快，同一赛事内不同赛项的队伍人数也可能不同。

## 一、核心赛事

|赛事|主要侧重|典型交付|官方入口|
|---|---|---|---|
|全国大学生机器人大赛（CURC）|Robocon 等机器人对抗/任务|整机、现场任务、技术文档|[官网](http://www.cnrobocon.net/)|
|中国机器人大赛暨 RoboCup 机器人世界杯中国赛|RoboCup 体系、多机器人与 AI|仿真或实体机器人比赛|[官网](http://crc.drct-caa.org.cn/)|
|睿抗机器人开发者大赛（RAICOM）|机器人开发、编程与应用|按赛项完成开发和现场任务|[官网](https://www.robocom.com.cn/)|
|中国高校智能机器人创意大赛|机器人创意与工程实现|创意作品、实物、报告和答辩|[官网](http://www.robotcontest.cn/)|
|中国机器人及人工智能大赛|机器人、AI 和交叉应用|竞技、算法或创新作品|[官网](https://www.caairobot.com/)|

以上赛事均列入 2024《全国普通高校大学生竞赛分析报告》目录，但具体子赛、专项赛是否计入学校同一认定等级，以本校文件为准。

## 二、赛事差异

### 全国大学生机器人大赛（CURC）

目录名称已将原先分别统计的 RoboMaster、Robocon 等整合为 CURC。实际参赛时仍需进入具体子赛规则；不同子赛在机器人数量、场地、对抗、操作方式、经费和安全方面差异很大。

### 中国机器人大赛暨 RoboCup 中国赛

通常包含足球、服务、救援、仿真等 RoboCup 相关组别。国际规则、中国赛补充规则和当届技术委员会解释应同时核对。是否构成 RoboCup 国际赛选拔，以具体组别通知为准。

### RAICOM

原 RoboCom 机器人开发者大赛已更名为睿抗机器人开发者大赛。任务可能偏编程、开发、数字孪生或实体机器人；不要用旧名称和旧组别推断当届规则。

### 中国高校智能机器人创意大赛

更强调创意作品和完整系统。除了展示功能，还应提交问题证据、系统架构、结构/电路/算法、测试、成本和应用边界。

### 中国机器人及人工智能大赛

当前官网按赛区和赛项发布通知。机器人竞技、无人驾驶、AI 创新等方向可能并行，报名入口和晋级路径以当届赛项页为准。

## 三、技术栈选择

|层级|常见选择|验收重点|
|---|---|---|
|机械|轮式、足式、机械臂、云台、执行器|强度、刚度、间隙、重量、维护和安全|
|电气|电源、驱动、传感、保护、通信|功耗、EMI、过流/反接、接插件可靠性|
|控制|PID、状态估计、轨迹规划、MPC|模型、频率、稳定性、扰动和饱和|
|感知|相机、激光、IMU、编码器、融合|标定、时钟、遮挡、噪声和失效模式|
|决策|状态机、搜索、优化、强化学习|实时性、可解释、回退和安全边界|
|系统|ROS/ROS 2、RTOS、嵌入式 Linux|版本、网络、启动、日志和可复现部署|

规则指定控制器、车模、算力、传感器或禁用器件时，指定规则高于技术偏好。

## 四、仿真到实机

1. 先建立最小可运行链路：输入、状态、决策、控制、执行和日志。
2. 仿真中加入噪声、延迟、摩擦、丢包、遮挡和执行器限制。
3. 实机分模块测试，不在完整机器人上同时调所有问题。
4. 使用统一坐标系、时间戳、单位和配置管理。
5. 每次场测保存软件版本、参数、地图/场地、视频和日志。
6. 建立降级模式：感知失效、网络中断、低电量、过温和急停。

## 五、安全与合规

- 机械运动范围设置防护，调试时使用支架、限速和物理急停。
- 电池使用合适充电器、保险和防火容器；运输遵守场馆与交通要求。
- 无人机、无线频段、激光和高功率设备按法规和场馆规则使用。
- 视觉数据涉及人脸、车牌或室内隐私时脱敏并限制留存。
- 不在比赛期间共享赛题、地图、Flag、对手代码或保密资料。
- 开源 ROS 包、模型和数据集需核对许可证和署名要求。

## 六、赛前检查

- [ ] 机器人从完全断电状态可在规定时间内启动。
- [ ] 参数、模型、固件和机械版本有唯一编号。
- [ ] 备用电池、控制器、传感器、线缆和紧固件齐全。
- [ ] 网络断开、传感器失效和执行器堵转有回退。
- [ ] 队员都能安全接管、急停、换件和恢复系统。
- [ ] 现场尺寸、光照、地面、无线环境和电源已模拟。
- [ ] 技术报告中的性能来自重复测试，不是单次最佳视频。
