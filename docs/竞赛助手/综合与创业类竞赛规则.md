# 综合与创业类竞赛规则知识库

## 中国国际 “互联网+” 大学生创新创业大赛

- **竞赛分类**：综合类

- **主办单位**：教育部、中央统战部、中央网络安全和信息化委员会办公室、国家发展和改革委员会、工业和信息化部、人力资源和社会保障部、农业农村部、商务部、国家卫生健康委员会、国家知识产权局、中国科学院、中国工程院、国务院扶贫开发领导小组办公室、共青团中央等

- **参赛对象**：普通高等学校、职业院校（含高职、中职）全日制在校大学生、毕业五年内的高校毕业生（含留学回国人员）、港澳台地区相关高校学生，以及在华国际留学生均可参赛。分为高教主赛道、“青年红色筑梦之旅”赛道、职教赛道、国际赛道和萌芽赛道（中小学生参与）。

- **报名时间**：约每年4月启动报名，截止至5月

- **比赛时间**：约每年5月—10月，分为校级初赛、省级复赛、全国总决赛三个阶段

- **赛道设置**：
        高教主赛道：分为创意组、初创组、成长组、产业命题组，涵盖人工智能、大数据、云计算、生物医药、智能制造等多个领域

- “青年红色筑梦之旅”赛道：分为公益组、商业组，聚焦乡村振兴、精准扶贫、社区服务等社会问题

- 职教赛道：突出职业教育特色，分为创意组、创业组、产业命题组，强调产教融合、校企合作

- 国际赛道：面向全球高校学生，促进中外青年交流合作

- 萌芽赛道：面向中小学生，培养创新意识和创业思维

**评审标准**：主要从项目创新性、商业性、团队情况、社会价值、发展前景等方面进行评审。其中，高教主赛道侧重商业潜力和技术创新，“青年红色筑梦之旅”赛道侧重社会贡献和公益价值。

**奖项设置**：设金奖、银奖、铜奖，各赛道分别评选。同时设置最佳创意奖、最具商业价值奖、最佳带动就业奖等单项奖，以及优秀组织奖、优秀指导教师奖等。

**关键特点**：教育部主办，是目前国内覆盖面最广、参赛人数最多、影响力最大的顶级创新创业赛事，被誉为“双创教育的奥林匹克盛会”，对大学生创新创业能力培养和项目孵化具有重要推动作用。

## “挑战杯” 全国大学生课外学术科技作品竞赛

- **竞赛分类**：综合类

- **主办单位**：共青团中央、中国科协、教育部、全国学联、地方人民政府

- **参赛对象**：全日制非成人教育的各类高等院校在校专科生、本科生、硕士研究生和博士研究生（均不含在职研究生）均可申报作品参赛。以学校为单位组织申报，实行校、省、全国三级竞赛制度。

- **作品类别**：
        自然科学类学术论文：包括数学、物理、化学、天文、地理、生物等学科的理论研究和实验研究论文

- 哲学社会科学类社会调查报告和学术论文：包括哲学、经济、法律、教育、管理、文学、历史、新闻传播学等学科的调查报告和学术论文

- 科技发明制作类：分为A类（科技含量较高、制作投入较大的作品）和B类（投入较少，且为生产技术或社会生活带来便利的小发明、小制作）

**报名时间**：偶数年约11 - 12月完成省级作品申报

**比赛时间**：偶数年约6月—8月举行全国总决赛，此前依次进行校级初赛和省级复赛

**评审标准**：自然科学类论文侧重学术性、科学性、创新性；哲学社会科学类作品侧重理论意义、应用价值、现实针对性；科技发明制作类侧重发明的新颖性、创造性、实用性及经济效益、社会效益。

**奖项设置**：设特等奖、一等奖、二等奖、三等奖，各类别分别评选。同时设置累进创新奖、交叉创新奖、最具转化潜力奖等专项奖，以及优秀组织奖。

**关键特点**：俗称“大挑”，自1989年创办以来，已成为全国大学生学术科技领域的最高赛事，注重学术科技作品的原创性和研究深度，是检验高校学术育人成果的重要平台，被称为全国大学生学术科技的“奥林匹克”。

## “挑战杯” 中国大学生创业计划大赛

- **竞赛分类**：综合类

- **主办单位**：共青团中央、教育部、人力资源和社会保障部、中国科协、全国学联、地方人民政府

- **参赛对象**：全日制普通高等学校在校专科生、本科生、硕士研究生、博士研究生（均不含在职研究生），毕业五年内（含毕业当年）的高校毕业生（含留学回国人员、港澳台地区毕业生）也可参赛。以团队形式报名，每个团队人数不超过15人（含指导教师）。

- **赛道设置**：
        主体赛：分为创新组、创业组、就业组，涵盖新一代信息技术、高端装备制造、新能源新材料、生物医药、节能环保、文化创意和现代服务业等领域

- 红旅赛：分为公益组、商业组，聚焦乡村振兴、红色文化传承等主题

- 产业命题赛道：由企业提出实际需求命题，参赛团队围绕命题开展创业计划设计

**报名时间**：奇数年约11 - 12月完成省级作品申报

**比赛时间**：奇数年约3月—5月举行全国总决赛，前期为校级初赛和省级复赛

**评审标准**：主要考察项目的商业计划书质量、产品或服务创新性、市场分析与定位、商业模式设计、团队能力、财务规划、社会价值及落地可行性等。

**奖项设置**：设金奖、银奖、铜奖，各赛道分别评选。还设有最佳创意奖、最具商业价值奖、最佳带动就业奖等单项奖，以及优秀组织奖、优秀指导教师奖。

**关键特点**：俗称“小挑”，与“大挑”隔年举办，侧重于商业计划书的评审与项目落地，强调创业实践能力和市场运作能力，是培养大学生创业精神、提升创业技能的重要赛事，也是创业项目对接资本和市场的重要桥梁。

## 全国大学生创新创业训练计划年会展示

- **竞赛分类**：综合类

- **主办单位**：教育部高等教育司

- **参赛对象**：入选国家级大学生创新创业训练计划（包括创新训练项目、创业训练项目、创业实践项目）的优秀项目团队，由各高校推荐报送。

- **项目类型**：
        创新训练项目：本科生个人或团队在导师指导下，自主完成创新性研究项目设计、研究条件准备和项目实施、研究报告撰写、成果（学术）交流等工作

- 创业训练项目：本科生团队在导师指导下，基于一项具有市场前景的技术产品或服务，完成商业计划书编制、可行性研究、企业模拟运行等工作

- 创业实践项目：本科生团队在学校导师和企业导师共同指导下，采用前期创新训练项目（或创新性实验）的成果，开展创业实践活动

**报名时间**：约每年3月底之前，各高校完成推荐项目报送

**展示形式**：采用线上线下相结合的方式，包括项目展板展示、现场路演、专家提问点评等环节，部分优秀项目还会进行成果转化交流。

**评审标准**：重点考察项目的创新性、完成质量、学生自主参与程度、实践能力提升效果以及成果的应用价值和推广前景等。

**奖项设置**：评选优秀组织单位、优秀项目奖（分为一等奖、二等奖、三等奖）、最佳创意奖、最具转化价值奖等。

**关键特点**：依托国家级大学生创新创业训练计划，是对高校创新创业教育实践成果的集中展示和交流平台，旨在促进项目成果的推广与转化，激发大学生的创新思维和创业热情。

## 中美青年创客大赛

- **竞赛分类**：综合类

- **主办单位**：教育部留学服务中心、美国高通公司等

- **参赛对象**：中美两国16-45岁的青年创客（包括大学生、创业者、设计师、工程师等），以团队形式参赛，每支团队人数3-5人，可跨国家、跨学科组队。

- **竞赛主题**：每年设定不同主题，聚焦社会热点问题，如“共创未来”“可持续发展”“智慧生活”等，要求参赛作品具有创新性、实用性和社会价值。

- **报名时间**：约每年6月1日 - 7月20日

- **比赛时间**：约每年6月1日-7月20日为报名及作品提交阶段，之后进行区域选拔赛和全国总决赛，总决赛通常在8-9月举行

- **作品要求**：需提交原创的产品原型或解决方案，包括设计方案、技术文档、演示视频等，强调作品的可实现性和市场应用潜力。

- **评审标准**：从创新性（技术或设计创新）、实用性（解决实际问题的能力）、团队协作（跨文化、跨学科合作）、展示效果（方案呈现和现场答辩）等方面进行评审。

- **奖项设置**：设一等奖、二等奖、三等奖及单项奖（如最佳创新奖、最佳设计奖等），获奖团队可获得奖金、创业孵化支持、国际交流机会等。

- **关键特点**：中美两国青年交流的重要平台，以“创新、合作、发展”为核心，促进中美青年在科技、文化、创业等领域的交流与合作，培养具有国际视野的创新人才。

## 全国大学生能源经济学术创意大赛

- **竞赛分类**：综合类

- **主办单位**：中国“双法”研究会能源经济与管理研究分会、教育部高等学校能源动力类专业教学指导委员会等

- **参赛对象**：全国普通高等院校全日制在读本科生、硕士研究生、博士研究生（均不含在职研究生），以个人或团队形式参赛，团队人数不超过5人（含指导教师1-2人）。

- **竞赛内容**：
        学术论文类：围绕能源经济、能源政策、能源市场、能源安全、能源与环境协调发展等主题展开的理论研究或实证分析论文

- 创意作品类：包括能源经济相关的创意方案、数据可视化作品、政策建议报告等，要求具有创新性和实践应用价值

**报名时间**：约每年10月17日 - 11月16日

**比赛时间**：约每年10月—11月为作品提交阶段，12月左右举行全国总决赛，总决赛包括作品展示、现场答辩等环节

**评审标准**：学术论文类侧重学术性、科学性、创新性和逻辑性；创意作品类侧重创意性、实用性、可行性及对能源经济问题的解决效果。

**奖项设置**：设一等奖、二等奖、三等奖、优秀奖，以及最佳指导教师奖、优秀组织单位奖等。

**关键特点**：聚焦能源经济领域，是全国首个以能源经济为主题的大学生学术创意竞赛，旨在培养大学生对能源经济问题的研究能力和创新思维，为能源领域的发展输送优秀人才。

## iCAN 大学生创新创业大赛

- **竞赛分类**：综合类

- **主办单位**：国际iCAN联盟、教育部创新方法教学指导分委员会等

- **参赛对象**：全国普通高等院校全日制在读本科生、专科生、研究生（含硕士、博士研究生），以团队形式参赛，每支团队人数2-5人，可配备1-2名指导教师。

- **竞赛理念**：以“创新引领未来，创业成就梦想”为理念，强调“源于教育、成于创新、归于创业”，鼓励大学生将创新想法转化为实际产品。

- **作品范围**：涵盖智能硬件、物联网、人工智能、机器人、生物医药、新材料、节能环保、文化创意等多个领域，要求作品具有原创性、创新性和实用性，能够解决实际问题或满足市场需求。

- **报名时间**：约每年7月31日前完成省级赛区报名及作品提交

- **比赛时间**：约每年7月31日前为省级赛区比赛阶段，8-9月举行全国总决赛

- **评审标准**：主要从作品的创新性（核心技术或设计创新）、实用性（市场需求和应用前景）、技术难度（技术实现的复杂性和先进性）、团队表现（现场答辩和作品演示）等方面进行评审。

- **奖项设置**：全国总决赛设金奖、银奖、铜奖，省级赛区设一、二、三等奖及优秀奖。获奖作品可获得创业孵化支持、投融资对接机会、国际赛事推荐资格等。

- **关键特点**：起源于2007年，是一项具有国际影响力的创新创业赛事，注重培养大学生的创新实践能力和工程应用能力，搭建了高校与企业之间的产学研合作桥梁，推动创新成果的转化和应用。

所有竞赛的具体时间、赛道设置、评审标准等每年可能会根据实际情况进行微调，请务必以各赛事官方网站发布的最新通知为准，及时关注报名及比赛动态。
