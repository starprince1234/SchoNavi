# 常见问题解答（FAQ）

## 1. 比赛基本信息类

问：挑战杯和互联网+有什么区别？
答：挑战杯更侧重于学术科研和技术创新，分为"大挑"（学术科技作品竞赛）和"小挑"（创业计划竞赛）。互联网+更侧重于创新创业项目的落地和商业化，尤其关注互联网技术与传统产业的结合。简单说：挑战杯偏学术科研，互联网+偏商业落地。

问：大创项目是什么？和竞赛有什么关系？
答：大创（大学生创新创业训练计划）是一个长期的训练项目，通常持续1-2年，有经费支持，重点是过程培养。大创的优秀成果可以作为参加挑战杯、互联网+等竞赛的基础。竞赛则是阶段性成果的评比和展示。

问：参加竞赛一定要有指导老师吗？
答：绝大多数比赛都要求有指导老师，特别是校级以上的比赛。指导老师的作用：提供专业指导、帮助联系资源、把关项目方向。建议提前联系相关领域老师，最好是有竞赛指导经验的。

问：一个人可以参加比赛吗？
答：大部分比赛允许个人参赛，但强烈建议组队参加。因为：
1. 比赛涉及多方面能力（技术、文案、演讲、美工）
2. 团队可以分工合作，提高效率
3. 评审看重团队的综合能力
4. 队友可以互相督促和支持

## 2. 组队与团队类

问：如何找到合适的队友？
答：有效找队友的方法：
1. 校内渠道：相关学院布告栏、实验室、学生社团、课程项目同学
2. 线上平台：学校竞赛群、相关技术社区、竞赛平台
3. 老师推荐：指导老师通常认识有能力的同学
4. 比赛说明会：很多比赛会组织组队交流会

问：一个理想的竞赛团队需要哪些角色？
答：核心角色包括：
1. 项目负责人：整体协调、进度管理、对外联系
2. 技术核心：负责技术实现、算法开发、系统搭建
3. 商业/市场：市场分析、商业模式、财务预测
4. 设计/美工：UI设计、PPT美化、视频制作
5. 演讲展示：路演答辩、文案撰写、沟通表达

问：队友中途退出怎么办？
答：预防和应对措施：
1. 提前明确团队规则和 commitment
2. 任务分解，每人有明确职责
3. 定期开会同步进度
4. 如果确实有人退出，及时寻找替补或调整分工
5. 联系指导老师或竞赛组委会说明情况

## 3. 时间与准备类

问：一般需要提前多久准备竞赛？
答：建议时间规划：
- 简单校级比赛：1-2个月
- 省级比赛：3-6个月
- 国家级比赛：6-12个月
例如互联网+大赛：1-2月构思组队，3-4月完善材料，5-6月校内赛，7-8月省赛，9-10月国赛。

问：大二开始准备竞赛晚吗？
答：完全不晚！大二是参加竞赛的黄金时间：
1. 有了一定的专业基础
2. 有充足的时间投入
3. 可以连续参加2-3年
4. 大二打好基础，大三冲刺高级别奖项

问：同时参加多个比赛可以吗？
答：可以，但要合理规划：
1. 选择1-2个主攻比赛
2. 确保项目有通用性（一个项目可参加多个比赛）
3. 合理安排时间，避免冲突
4. 团队成员分工协调

## 4. 材料准备类

问：商业计划书必须写财务预测吗？
答：对于商业类竞赛（如互联网+），财务预测是必须的，应包括：
1. 未来3年收入预测（按月/季）
2. 成本构成（研发、运营、市场、人力）
3. 利润预测
4. 关键财务假设（用户增长、客单价等）
即使项目早期，也要有合理的估算逻辑。

问：答辩PPT一般多少页合适？
答：建议结构：
1. 封面（1页）：项目名称、团队、学校
2. 痛点与市场（2-3页）
3. 解决方案（2-3页）
4. 产品与技术（3-4页）
5. 商业模式（2-3页）
6. 团队介绍（1-2页）
7. 财务与规划（2-3页）
8. 结尾与Q&A（1页）
总计15-20页，时间控制在8-10分钟。

问：技术类作品需要做出实物吗？
答：根据比赛要求：
1. 如果能做出实物或demo，是很大的加分项
2. 如果条件有限，要有详细的技术方案和仿真结果
3. 至少要有完整的设计图纸、算法流程图、测试数据
4. 视频演示效果也很好

## 5. 答辩展示类

问：答辩时评委最关注什么？
答：不同比赛侧重点不同：
- 学术类：创新性、科学性、完整性
- 创业类：市场潜力、商业模式、团队能力
- 技术类：技术难度、实用性、成熟度
共同关注：项目价值、逻辑严谨、团队配合。

问：如何应对答不上来的问题？
答：应对策略：
1. 诚实承认：对不了解的部分诚实说明
2. 转移焦点：引导到自己熟悉的方面
3. 承诺改进：表示会后研究完善
4. 团队补充：让了解的队友回答
绝对不要：不懂装懂、争论不休。

问：路演时紧张怎么办？
答：缓解紧张的方法：
1. 充分准备：熟记内容，多次演练
2. 提前适应：在比赛场地提前练习
3. 心理暗示：把评委当朋友，分享你的成果
4. 团队配合：队友可以补充
5. 关注内容：专注于你要传达的信息
