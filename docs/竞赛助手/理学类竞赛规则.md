# 理学类竞赛规则知识库

本文档汇总了国内主流理学类大学生竞赛的详细规则信息，包括竞赛简介、参赛对象、报名与比赛时间、竞赛内容及奖项设置等，为相关专业学生参赛提供全面参考。

## 全国大学生数学建模竞赛

- **竞赛分类**：理学类

- **竞赛简介**：全国大学生数学建模竞赛是由教育部高等教育司、中国工业与应用数学学会联合主办的全国性高校学科竞赛，创办于1992年，是首批列入“高校学科竞赛排行榜”的竞赛项目。竞赛旨在培养学生的创新意识、团队协作能力和解决实际问题的能力，通过构建数学模型解决现实生活中的实际问题。

- **参赛对象**：全日制普通高等院校本科及专科学生（含高职高专学生）均可组队参赛，每支队伍由3名学生组成，可跨专业、跨年级组队。

- **报名时间**：约每年**8月**，具体时间以各赛区组委会通知为准，参赛队伍需通过所在学校统一报名。

- **比赛时间**：约每年**9月第二个周四至周日**（原信息中9月4日为大致时间，实际通常为该时间段内的4天3夜），比赛采用通讯赛形式，参赛队伍在规定时间内完成论文提交。

- **竞赛内容**：竞赛题目通常来源于工程技术、管理科学、社会经济等领域的实际问题，不指定具体参考书和工具。参赛队伍需在4天内完成问题分析、模型构建、算法设计、编程实现、结果分析及论文撰写等工作，提交一篇完整的论文。

- **奖项设置**：竞赛设全国一、二等奖（国家级奖项），各赛区设一、二、三等奖（省级奖项）。国家级奖项获奖比例约为全国参赛队伍总数的5%（一等奖约1%，二等奖约4%）。

## 全国周培源大学生力学竞赛

- **竞赛分类**：理学类

- **竞赛简介**：全国周培源大学生力学竞赛是为纪念我国著名力学家、教育家周培源先生而设立的全国性力学学科竞赛，始于1986年，由教育部高等学校力学基础课程教学指导委员会、中国力学学会主办。竞赛旨在激发大学生学习力学的兴趣，培养其力学分析与解决问题的能力，推动力学基础课程教学改革。

- **参赛对象**：全日制普通高等院校本科二年级及以上相关专业（如机械、土木、航空航天、力学等）学生，以个人名义参赛。

- **报名时间**：约每年**4月10日**前，由各高校统一组织报名，学生需在学校规定时间内提交报名信息。

- **比赛时间**：约每年**5月下旬**（原信息中比赛时间与报名时间均为4月10日前有误，实际报名在4月前，比赛在5月），比赛形式为闭卷笔试，时长3小时。

- **竞赛内容**：主要涵盖理论力学和材料力学的基本内容，包括静力学、运动学、动力学、强度、刚度、稳定性等。题目注重考查学生对力学基本概念、基本原理的理解及应用能力，部分题目涉及工程实际背景。

- **奖项设置**：竞赛设全国一、二、三等奖及优秀奖，各赛区也设相应省级奖项。全国一等奖获奖比例约为全国参赛人数的1%，二等奖约3%，三等奖约6%。

## 全国大学生物理实验竞赛

- **竞赛分类**：理学类

- **竞赛简介**：全国大学生物理实验竞赛是由教育部高等学校物理学类专业教学指导委员会主办的全国性实验技能竞赛，始于2010年，旨在加强大学生物理实验能力的培养，促进高校物理实验教学改革与交流，提高学生的创新思维和实践操作水平。

- **参赛对象**：全日制普通高等院校本科在读学生，以团队形式参赛，每支队伍由3名学生和1-2名指导教师组成。

- **报名时间**：约每年**9月1 - 25日**，参赛队伍需通过学校推荐，登录竞赛官方平台提交报名材料。

- **比赛时间**：约每年**11月中下旬**（原信息中比赛时间9月1日—9月15日有误，实际报名在9月，省级比赛在10月，全国决赛在11月），比赛分为省级预赛和全国决赛，均为实验操作和报告撰写相结合的形式。

- **竞赛内容**：包括基础性实验、综合性实验和设计性/研究性实验三个类别。基础性实验考查基本实验技能，综合性实验考查实验设计与数据分析能力，设计性/研究性实验考查创新思维与科研潜力，题目涵盖力学、热学、电磁学、光学等多个物理分支。

- **奖项设置**：全国决赛设一、二、三等奖及单项奖，省级预赛设相应省级奖项。全国一等奖获奖比例约为全国决赛参赛队伍的15%，二等奖约30%，三等奖约40%。

## 全国大学生化学实验创新设计大赛

- **竞赛分类**：理学类

- **竞赛简介**：全国大学生化学实验创新设计大赛是由教育部高等学校化学类专业教学指导委员会、高等学校国家级实验教学示范中心联席会主办的全国性化学实验竞赛，始于2018年，旨在激发大学生对化学实验的兴趣，培养其创新能力、实践能力和团队协作精神，推动化学实验教学改革。

- **参赛对象**：全日制普通高等院校化学、应用化学、化学工程与工艺等相关专业本科在读学生，以团队形式参赛，每支队伍由3-5名学生和1-2名指导教师组成。

- **报名时间**：约每年**6月**，各高校先组织校内选拔，推荐优秀队伍参加省级初赛，省级初赛优秀队伍晋级全国决赛。

- **比赛时间**：约每年**8月下旬至9月上旬**（原信息中比赛时间与报名时间均为6月有误，实际报名在6月，省级比赛在7月，全国决赛在8-9月），比赛分为实验设计方案汇报、实验操作展示和现场答辩三个环节。

- **竞赛内容**：要求参赛队伍围绕化学实验教学、绿色化学、新材料合成、环境治理等领域设计创新性实验方案，方案需具有科学性、创新性、可行性和教学适用性。比赛中需展示实验操作过程，并对实验结果进行分析与答辩。

- **奖项设置**：全国决赛设一、二、三等奖及最佳创新奖、最佳实践奖等单项奖，省级初赛设相应省级奖项。全国一等奖获奖比例约为全国决赛参赛队伍的20%。

## 全国大学生地质技能竞赛

- **竞赛分类**：理学类

- **竞赛简介**：全国大学生地质技能竞赛是由教育部高等学校地质学类专业教学指导委员会、中国地质学会主办的全国性地质学科竞赛，始于2010年，是地质学领域最具影响力的大学生竞赛之一。竞赛旨在培养大学生的地质实践技能和创新能力，促进地质教育的发展与人才培养。

- **参赛对象**：全日制普通高等院校地质学、资源勘查工程、地质工程等相关专业本科在读学生，以团队形式参赛，每支队伍由3-5名学生组成。

- **报名时间**：约每年**3月15日 - 4月20日**，参赛队伍需通过学校登录竞赛官方网站提交报名信息及相关材料。

- **比赛时间**：约每年**4月或5月**，比赛形式为野外实践操作与室内综合测试相结合，时长通常为3-4天。

- **竞赛内容**：主要包括地质填图、岩石矿物鉴定、地层古生物识别、地质工程勘察设计等模块。野外环节考查学生的地质现象观察、数据采集与分析能力；室内环节考查学生对地质标本的鉴定能力和综合地质问题的解决能力。

- **奖项设置**：竞赛设全国一、二、三等奖及优秀组织奖，各单项也设相应奖项。全国一等奖获奖比例约为全国参赛队伍的10%，二等奖约20%，三等奖约30%。

## 全国大学生生命科学竞赛（CULSC）

- **竞赛分类**：理学类

- **竞赛简介**：全国大学生生命科学竞赛（CULSC）是由教育部高等学校生物技术、生物科学类专业教学指导委员会主办的全国性生命科学领域竞赛，始于2017年，旨在激发大学生对生命科学的兴趣，培养其科研创新能力和实践能力，推动生命科学学科的发展。

- **参赛对象**：全日制普通高等院校生命科学相关专业（如生物科学、生物技术、生物工程等）本科在读学生，分为创新组和创业组，以团队形式参赛，每支队伍由3-5名学生和1-2名指导教师组成。

- **报名时间**：约每年**9月13日 - 11月30日**，参赛队伍需通过学校推荐，在竞赛官方平台提交项目申报书等材料。

- **比赛时间**：约每年**次年5月 - 6月**（原信息中比赛时间9—11月有误，实际报名在9-11月，省级比赛在次年3-4月，全国决赛在次年5-6月），比赛分为省级初赛和全国决赛，以项目汇报和答辩的形式进行。

- **竞赛内容**：创新组要求提交生命科学基础研究或应用基础研究类项目，创业组要求提交生命科学领域的创业项目。项目需具有科学性、创新性和应用前景，参赛队伍需展示项目的研究背景、方法、结果及创新点，并回答评委提问。

- **奖项设置**：全国决赛设一、二、三等奖及最佳创新奖、最佳创业奖等，省级初赛设相应省级奖项。全国一等奖获奖比例约为全国决赛参赛队伍的15%，二等奖约25%，三等奖约40%。

## 全国大学生统计建模大赛

- **竞赛分类**：理学类

- **竞赛简介**：全国大学生统计建模大赛是由教育部高等学校统计学类专业教学指导委员会、中国统计教育学会主办的全国性统计学科竞赛，始于2008年，旨在培养大学生的统计建模能力和数据分析能力，提高其运用统计方法解决实际问题的水平，促进统计教育的改革与发展。

- **参赛对象**：全日制普通高等院校本科及专科学生（含高职高专学生）均可组队参赛，每支队伍由2-3名学生组成，可跨专业、跨年级组队，指导教师不超过2名。

- **报名时间**：约每年**2月20日 - 3月31日**，参赛队伍需通过学校登录竞赛官方网站完成报名。

- **比赛时间**：约每年**5月 - 6月**（原信息中比赛时间2月—3月有误，实际报名在2-3月，比赛在5-6月），比赛采用通讯赛形式，参赛队伍在规定时间内完成统计建模论文的撰写与提交。

- **竞赛内容**：竞赛题目通常来源于经济、社会、科技、环境等实际领域的统计问题，提供相关数据或数据获取途径。参赛队伍需运用统计理论和方法，构建统计模型，对数据进行分析与预测，并撰写详细的建模论文，包括问题分析、模型假设、模型构建、模型检验、结果解释等内容。

- **奖项设置**：竞赛设全国一、二、三等奖及优秀组织奖，各赛区设相应省级奖项。全国一等奖获奖比例约为全国参赛队伍总数的3%，二等奖约7%，三等奖约15%。

## 全国大学生数学竞赛

- **竞赛分类**：数学类

- **竞赛简介**：全国大学生数学竞赛是由中国数学会主办的全国性数学学科竞赛，始于2009年，旨在培养大学生学习数学的兴趣，提高其数学素养和创新能力，发现和选拔数学创新人才。竞赛分为数学类和非数学类两个组别，其中数学类面向数学专业学生，非数学类面向其他专业学生。

- **参赛对象**：全日制普通高等院校本科在读学生，数学类专业学生参加数学类竞赛，非数学类专业学生参加非数学类竞赛，以个人名义参赛。

- **报名时间**：约每年**3月 - 6月**（不同赛区有差异），由各高校统一组织报名，学生需在学校规定时间内提交报名信息。

- **比赛时间**：约每年**10月**初赛，**11 - 12月**决赛。初赛为闭卷笔试，时长3小时；决赛难度高于初赛，同样为闭卷笔试，时长3小时。

- **竞赛内容**：数学类初赛内容包括数学分析、高等代数、解析几何等；决赛在初赛基础上增加常微分方程、概率论等内容。非数学类初赛内容包括高等数学、线性代数等；决赛在初赛基础上增加少量概率论与数理统计内容。题目注重考查学生对数学基本概念、基本理论和基本方法的掌握及灵活运用能力。

- **奖项设置**：初赛设省级一、二、三等奖；决赛设全国一、二、三等奖，其中数学类和非数学类分别评奖。全国一等奖获奖比例约为全国决赛参赛人数的10%，二等奖约20%，三等奖约30%。获得全国一等奖的学生有机会参加中国数学奥林匹克（CMO）的选拔。

**注意**：所有竞赛的具体时间、参赛要求及流程每年可能微调，以上信息仅供参考。请务必以各赛事官方网站或组委会发布的最新通知为准，避免因信息滞后影响参赛。
