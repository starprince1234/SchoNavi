# SchoNavi 接口文档

本文件已迁移为索引页，避免维护多份互相冲突的接口定义。

权威 contract：

- 人类可读文档：[`docs/api-contract.md`](docs/api-contract.md)
- 机器可读 OpenAPI：[`docs/openapi.yaml`](docs/openapi.yaml)

后端开发请以 `/api/v1` + envelope 响应格式为准：

```json
{ "code": 0, "message": "ok", "data": {} }
```

旧版 `/api/recommendations`、`/api/professors/{id}`、`/api/chat/messages`
直返 JSON 契约已废弃，不再作为新后端实现依据。
